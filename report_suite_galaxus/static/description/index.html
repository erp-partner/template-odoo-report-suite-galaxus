How it works

Each standard Odoo report is inherited. When enabled in Settings, the Galaxus-style template replaces the default; disable to use Odoo’s standard template.

  • Sales/Pro-forma: sale.report_saleorder_document
  • Purchase/RFQ: purchase.report_purchaseorder_document, purchase.report_purchasequotation_document
  • Delivery: stock.report_delivery_document
  • Invoice: account.report_invoice_document

Toggles in Settings > General Settings via ir.config_parameter such as report_suite_galaxus.use_sale_report, ...use_purchase_reports, ...use_delivery_reports.

Configuration

  1. Install the module.
  2. Open Settings > General Settings and enable the reports you want.
  3. (Optional) Choose the provided paperformat to provide the correct page margins.
  4. Print Sales, Purchase, Delivery, or Invoice to verify the new layout.
  5. Field names can change in the same Odoo Version, contact me if you have errors during report printing.

Compatibility
